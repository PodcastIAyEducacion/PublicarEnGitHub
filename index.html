Publica tu artefacto
de Claude en la web

Una guia paso a paso para publicar cualquier artefacto de Claude con una URL propia, gratuita y accesible desde cualquier dispositivo.

PodcastIAyEducacion marzo 2026 ~10 min la primera vez
LO QUE NECESITAS

Una cuenta gratuita en github.com y el codigo de tu artefacto (boton < > en Claude).

RESULTADO FINAL

Una URL publica del tipo:

podcastiayeducacion.github.io/nombre
PASO 1
Obtén el código del artefacto
  1. 1Abre la conversacion de Claude donde esta el artefacto.
  2. 2En la esquina superior derecha del artefacto, haz clic en el boton < > (ver codigo).
  3. 3Selecciona todo el texto y copialo (Ctrl+A y luego Ctrl+C).
PASO 2
Crea el repositorio en GitHub
  1. 1Ve a github.com/new
  2. 2Veras la pantalla Create a new repository dividida en dos bloques. Rellena asi:

► BLOQUE GENERAL (parte superior de la pantalla)

CampoQue poner
OwnerSelecciona tu usuario en el desplegable (ej: PodcastIAyEducacion)
Repository nameNombre corto sin espacios (ej: campeones-andalucia)
DescriptionOpcional · max. 350 caracteres

► BLOQUE CONFIGURATION (parte inferior de la pantalla)

CampoQue hacer
Visibility✓ PUBLIC — para que sea accesible desde internet
Add README✓ ON — activa el toggle
Add .gitignoreNo .gitignore — dejalo asi
Add licenseNo license — dejalo asi
  1. 3Haz clic en el boton verde Create repository al final de la pagina.
PASO 3
Sube el archivo HTML
  1. 1Dentro del repositorio, haz clic en Add file → Create new file.
  2. 2En el campo del nombre (arriba del area de texto) escribe exactamente:
index.html
  1. 3Pega todo el codigo del artefacto en el area de texto grande.
  2. 4Haz clic en el boton verde Commit changes.
  3. 5Ignora el campo extended description y confirma con otro Commit changes.
PASO 4
Activa GitHub Pages
  1. 1Haz clic en la pestana Settings (parte superior del repositorio).
  2. 2En el menu lateral izquierdo, haz clic en Pages.
  3. 3En el apartado Branch, selecciona main y haz clic en Save.
  4. 4Si el boton Save aparece deshabilitado, la configuracion ya esta guardada.
  5. 5Espera 1-2 minutos.
https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO
PASO 5
Comprueba que funciona
  1. 1Abre la URL en el navegador. Si ves tu actividad, esta publicada.
  2. 2El enlace oficial aparece en Settings → Pages como un recuadro verde: "Your site is live at..."
💡 Cada artefacto va en su propio repositorio. Asi cada uno tiene su URL limpia y no se mezclan entre si.
EXTRA
Como encontrar tus enlaces despues

La app de GitHub en el movil no muestra el enlace de Pages directamente. Guarda cada URL en tus notas del movil en el momento de publicar. La formula es siempre la misma:

https://podcastiayeducacion.github.io/ + nombre del repositorio
